# EagleVStream v2.1 PRO

## نص إشهاري للبرنامج (Arabic)

### **EagleVStream v2.1 PRO – رفيقك المثالي لتحميل الفيديو والصوت بأقصى جودة!**

**مقدمة:**
هل تبحث عن تجربة تحميل سلسة، سريعة، واحترافية؟ إليك **EagleVStream** في إصداره الأحدث **v2.1 PRO**، الأداة الأقوى المصممة خصيصاً لتلبية احتياجاتك في تحميل المحتوى من مختلف المنصات العالمية بواجهة عصرية وسهولة فائقة.

---

### **💎 أهم المميزات التي تجعل EagleVStream خيارك الأول:**

#### **1. جودة لا تضاهى (دعم الـ 4K):**
*   استمتع بتحميل فيديوهاتك المفضلة بوضوح مذهل يبدأ من **144p** ويصل إلى **4K (2160p)**.
*   ذكاء اصطناعي لاختيار أفضل جودة متاحة تلقائياً لضمان أفضل تجربة مشاهدة.

#### **2. تعدد الصيغ (فيديو وصوت):**
*   **للفيديو:** يدعم البرنامج أشهر الصيغ العالمية: **MP4, WebM, AVI, WMV**.
*   **للصوت:** استخرج الصوت بضغطة زر بصيغ **MP3, M4A, WAV** بجودة نقية جداً (192kbps للملفات الصوتية).

#### **3. نظام التحميل الذكي والمرن:**
*   **التحميل الجماعي:** أدخل قائمة روابط دفعة واحدة أو رابط قائمة تشغيل (Playlist) كاملة، وسيتولى البرنامج معالجتها بالتتابع.
*   **استكمال التحميل الذكي:** وداعاً لضياع البيانات! البرنامج يدعم ميزة **Resume** من آخر بايت تم تحميله في حال إغلاق البرنامج أو حدوث انقطاع.
*   **صمود الشبكة:** نظام فريد يكتشف انقطاع الإنترنت تلقائياً ويوقف التحميل مؤقتاً، ثم يستأنفه فور عودة الاتصال دون أي تدخل منك.

#### **4. واجهة مستخدم احترافية وعصرية:**
*   **ثيمات مخصصة:** اختر بين **الوضع الداكن (Dark Mode)** المريح للعين أو **الوضع الفاتح (Light Mode)** الساطع والنظيف.
*   **6 ألوان جذابة:** خصص مظهر البرنامج بلونك المفضل (البنفسجي، التيل، الوردي، الأزرق، البرتقالي، أو الأخضر).
*   **مؤشرات حية:** شريط تقدم متحرك بتأثير "الشيمر" (Shimmer) مع عرض دقيق لنسبة التحميل والسرعة في الوقت الفعلي.
*   **دعم كامل للغة العربية:** واجهة مصممة بالكامل لتدعم اللغة العربية (RTL) بشكل مثالي وسلس.

#### **5. أدوات إدارة وتحكم متقدمة:**
*   **سجل التحميلات (History):** تتبع جميع عملياتك السابقة (التاريخ، الرابط، الصيغة، والحالة) للرجوع إليها في أي وقت.
*   **كشف الملفات المكررة:** ذكاء برمجي يفحص جهازك ويحذرك إذا كان الملف موجوداً مسبقاً لتوفير مساحة التخزين.
*   **تخصيص المسار:** اختر مجلد الحفظ المفضل لديك أو استخدم المجلد الافتراضي للنظام بذكاء.
*   **تنبيهات ذكية:** إشعارات نظام فور اكتمال التحميل مع خيار فتح مجلد الملف تلقائياً.

#### **6. تكامل تقني عالي:**
*   يعتمد البرنامج على محرك **yt-dlp** المتطور لضمان دعم أوسع للمواقع العالمية.
*   نظام فحص آلي للاعتمادات (Python, FFmpeg) للتأكد من أن كل شيء يعمل بكفاءة 100%.

## Technical Features Overview (English)

*   **Engine:** Powered by `yt-dlp` for maximum site compatibility.
*   **Frontend:** Built with C# WPF for a native, responsive experience.
*   **Multi-Platform Support:** Compatible with Windows 7 and later.
*   **4K Video Support:** Downloads up to 2160p resolution.
*   **Audio Extraction:** Converts videos to MP3, M4A, or WAV.
*   **Batch Downloads:** Supports multi-line URL input and full playlists.
*   **Resilience:** Built-in auto-resume logic for network interruptions.
*   **Theming:** Dynamic Dark/Light mode switching with 6 accent colors.
*   **Localization:** Full Arabic (RTL) and English support.
*   **History Tracking:** Persistent JSON-based download history.
*   **FFmpeg Integration:** Seamlessly detects and uses FFmpeg for high-quality merging.
